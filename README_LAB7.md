# Лабораторная работа 7

Добавлен функционал редактирования и удаления записей в базе данных для информационной подсистемы интернет-магазина.

## Новые контроллеры

- EditCategoryServlet / DeleteCategoryServlet
- EditProductServlet / DeleteProductServlet
- EditOrderDetailsServlet / DeleteOrderDetailsServlet
- EditOrderServlet / DeleteOrderServlet

## Новые представления

- views/editcategory.jsp
- views/editproduct.jsp
- views/editorder-details.jsp
- views/editorder.jsp

## Проверка

Откройте страницы:

- http://localhost:8080/internet_store/category
- http://localhost:8080/internet_store/product
- http://localhost:8080/internet_store/order-details
- http://localhost:8080/internet_store/order

В таблицах появились рабочие ссылки редактирования и удаления.
